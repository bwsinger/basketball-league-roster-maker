# Basketball League Roster Maker

A web app for generating and managing balanced basketball league rosters.

## Status

Planning repository initialized. Implementation will be added in future commits.

## Planned Capabilities

- Import player data from JSON, CSV, or YAML
- Generate balanced teams automatically
- Manually adjust teams with drag-and-drop
- Export rosters for sharing
